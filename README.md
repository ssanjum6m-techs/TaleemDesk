# TaleemDesk 🏫

**Lesson Planner & School Documents Generator for Pakistani Schools**
By AI-TEG Academy — aitegacademy.com

- PCTB / SNC aligned lesson plans (Playgroup to Matric)
- 40+ school documents: certificates, notices, date sheets, board-pattern question papers
- English, Urdu, and Bilingual output
- All Punjab BISE boards + FBISE + AKU-EB + Cambridge sources

## Deploy on Vercel

1. Push this folder to a GitHub repository
2. Import the repo at vercel.com (framework auto-detected: Vite)
3. Add Environment Variable: `ANTHROPIC_API_KEY` = your key from console.anthropic.com
4. Deploy 🚀

## Local development

```
npm install
npm run dev
```
Note: the /api function only runs on Vercel (or with `vercel dev`).
